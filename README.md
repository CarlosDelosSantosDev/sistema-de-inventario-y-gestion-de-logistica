# SIGLI - Prototipo de Sistema de Logística, Monitoreo e Inventario 🚚📦

Este proyecto es un **MVP (Producto Mínimo Viable) Funcional** diseñado para la gestión logística de almacenes y operadores de transporte. El sistema integra una plataforma web de administración centralizada con una aplicación móvil dedicada para operadores en ruta.

###  Estado del Proyecto: Prototipo Funcional / MVP
> **Nota de desarrollo:** Toda la arquitectura de software backend, la estructura de la base de datos, la interfaz web y la aplicación móvil se encuentran **100% desarrolladas y funcionales a nivel lógico**. La integración física con hardware (drones automatizados) y el consumo de servicios de mapas de producción se mantienen documentados teóricamente dentro del alcance técnico por limitaciones de presupuesto escolar y de hardware.

###  Características y Módulos Desarrollados

* **Módulo Administrativo (Plataforma Web):** Control total del inventario de almacenes, monitoreo lógico de contenedores y desarrollo del flujo para la activación simulada de un dron autónomo que localiza mercancía mediante el escaneo de códigos QR.
* **Módulo del Operador (Aplicación Móvil):** Diseñado para conductores de transporte de carga. Permite la gestión y visualización de viajes asignados, control del inventario cargado en el camión y un panel de simulación de ruta que integra consultas de clima en tiempo real mediante APIs.

###  Tecnologías Utilizadas

* **Backend & API:** Laravel (PHP)
* **Desarrollo Móvil:** Flutter (Dart)
* **Base de Datos:** MySQL
* **Virtualización e Infraestructura:** Docker y Nginx
